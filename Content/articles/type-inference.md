# Type Inference 

Swift can figure out what sort of type the variable using type inference. For example

```swift
  let str = "This is Swift code."
  print(str)
  let x = 1;
  x += 2
```

Becomes

```swift
  let str = "This is Swift code."
  print(str)
  let x = 1;
  x += 2
```
